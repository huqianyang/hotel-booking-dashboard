# 酒店预订数据可视化分析与预测系统

本仓库用于 Python 数据可视化期末课程设计，当前阶段提供 Flask 项目骨架与 GitHub 协作基线。

## 快速开始

```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
python run.py
```

## 测试

```bash
python -m pytest -v
```

## 目录说明

- `app/`：Flask 应用代码、页面模板和静态资源。
- `tests/`：自动化测试。
- `任务文档/`：需求、规划、界面原型和实现计划文档。
- `老师给的期末要求/`：课程要求与老师补充材料。
